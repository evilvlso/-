## myqueue.py
>>python实现的一个优先级队列，用法：
```python
p=PriorityQueue()
p.push(item,9)
p.push(item,6)
p.push(item,10)
p.pop()  # 10
p.pop()  # 9
p.pop()  # 6
```
>>数字越大优先级越高。
